用于打开 Material Design 模态对话框的服务。

Service to open Material Design modal dialogs.

名称 描述

当所有打开的对话框都关闭的时候会发出通知的流。如果没有打开的对话框，就会在订阅时立即触发。

Stream that emits when all open dialog have finished closing. Will emit on subscribe if there are no open dialogs to begin with.

link 类

link MatDialogConfig

使用 MatDialog 服务打开的模态对话框的配置

Configuration for opening a modal dialog with the MatDialog service.

属性
名称 描述

ariaDescribedBy: string | null

描述该对话框元素的 ID。

ID of the element that describes the dialog.

ariaLabel: string | null

要分配给对话框元素的 Aria 标签。

Aria label to assign to the dialog element.

ariaLabelledBy: string | null

标记该对话框的元素的 ID。

ID of the element that labels the dialog.

autoFocus: boolean

打开对话框时是否应该让第一个可获取焦点的元素获得焦点。

Whether the dialog should focus the first focusable element on open.

backdropClass: string | string[]

背景板的自定义类。

Custom class for the backdrop.

closeOnNavigation: boolean

当用户在历史记录中后退时，该对话框是否会关闭。请注意，这通常不包括单击链接（除非用户正在使用 HashLocationStrategy ）。

Whether the dialog should close when the user goes backwards/forwards in history. Note that this usually doesn't include clicking on links (unless the user is using the HashLocationStrategy).

componentFactoryResolver: ComponentFactoryResolver

备用 ComponentFactoryResolver，用于解析其关联组件。

Alternate ComponentFactoryResolver to use when resolving the associated component.

data: D | null

要注入到子组件中的数据。

Data being injected into the child component.

direction: Direction

对话框内容的布局方向

Layout direction for the dialog's content.

disableClose: boolean

用户是否可以使用 escape 或单击背景板来关闭该模态框。

Whether the user can use escape or clicking on the backdrop to close the modal.

hasBackdrop: boolean

对话框是否有背景板。

Whether the dialog has a backdrop.

height: string

该对话框的高度。

Height of the dialog.

id: string

该对话框的 ID。如果省略，就会生成一个唯一的。

ID for the dialog. If omitted, a unique one will be generated.

maxHeight: number | string

对话框的最大高度。如果提供了数字，则假设单位是像素。

Max-height of the dialog. If a number is provided, assumes pixel units.

maxWidth: number | string

对话框的最大宽度。如果提供了数字，则假设单位是像素。默认为 80vw。

Max-width of the dialog. If a number is provided, assumes pixel units. Defaults to 80vw.

minHeight: number | string

对话框的最小高度。如果提供了数字，则假设单位是像素。

Min-height of the dialog. If a number is provided, assumes pixel units.

minWidth: number | string

对话框的最小宽度。如果提供了数字，则假设单位是像素。

Min-width of the dialog. If a number is provided, assumes pixel units.

panelClass: string | string[]

浮层面板的自定义类。

Custom class for the overlay pane.

position: DialogPosition

位置改写。

Position overrides.

restoreFocus: boolean

该对话框是否应该在关闭之前将焦点还给以前拥有焦点的元素。

Whether the dialog should restore focus to the previously-focused element, after it's closed.

role: DialogRole

该对话框元素的 ARIA 角色。

The ARIA role of the dialog element.

scrollStrategy: ScrollStrategy

用于对话框的滚动策略。

Scroll strategy to be used for the dialog.

viewContainerRef: ViewContainerRef

附加到组件应该位于 Angular 的逻辑组件树中。这会影响注入时的可用内容以及在对话框中实例化的组件的变更检测顺序。这不会影响对象内容的渲染位置。

Where the attached component should live in Angular's logical component tree. This affects what is available for injection and the change detection order for the component instantiated inside of the dialog. This does not affect where the dialog content will be rendered.

width: string

该对话框的宽度。

Width of the dialog.
